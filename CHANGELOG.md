# Changelog

## 1.0.1 (2026-02-05)

- Add template preset `wuxia` (Chinese Wuxia / Jianghu aesthetic).
- Add `list_xhs_options` tool and built-in option pools for random variation.

- Add `list_xhs_templates` tool and template file support via `XHS_TEMPLATES_PATH`.
